# LLM Self-Explanations and Faithfulness


## Environment
Run `conda env create -f environment.yml`
Then `conda activate faithfulness-env`.

## Project Structure

### Core Directories

- **src/** - Core library code for experiments

  - `schema.py` - Data structures for storing experimental results
  - `utils.py` - Shared utilities (parsing, normalization, LLM configuration)
  - `templates/` - Dataset-specific prompt templates (breast cancer, heart disease, diabetes)
  - `experiments/` - Main experiment runners (multi-model analysis, simulatability tests)
  - `counterfactual_generation/` - Counterfactual generation logic
  - `prediction_generation/` - Predictor model answer generation
  - `reference_answer_generation/` - Reference model answer generation

- **analysis_scripts/** - python scripts for analyzing experimental results and plotting

- **run_scripts/** - bash execution scripts for running experiments

## Natural Counterfactual Dataset Set-Up
To generate Hammning-ball style counterfactual datasets, run
`python -m src.counterfactual_generation.tabular_counterfactual_generation.tabular_to_text --output_dir data/natural_counterfactuals` from project root.
To generate moral machines counterfactual dataset, run
`PYTHONHASHSEED=0 python -m src.counterfactual_generation.tabular_counterfactual_generation.moral_machines_generator`.
To build combined dataset, with samples from all of the above:
`python -m data.natural_counterfactuals.generate_combined`




# Schema

- dataset [str] # the dataset which the original_question comes from.

- original_question [str] # the original question
- original_question_prompt # entire prompt containing the original question
- original_question_idx [int] # a unique identifier for questions
- original_question_ground_truth [optional, str] # the ground truth for the original question

- counterfactual_generator_model [str]
- counterfactual_generator_method [str]

- counterfactual_question [str] # a counterfactual generated based on the original_question.
- counterfactual_question_prompt [str] # a counterfactual generated based on the original_question. This should be the entire prompt for the counterfactual
- counterfactual_answer_first [optional, Bool] # a Boolean representing whether or not the counterfactual was generated using an explanation which came before or after the answer to the original question
- counterfactual_idx [int] # a unique identifier for counterfactuals for a particular original_question
- counterfactual_ground_truth [optional, str] # ground truth values for the counterfactuals

- counterfactual_coherence_scored_by_generator [optional, Bool] # coherence (True/False) as judged by model during counterfactual generation
- counterfactual_coherence_explanation_scored_by_generator [optional, str] # explanation of coherence score given by model during counterfactual generation
- counterfactual_coherence_external_scoring_model [optional, str] # model used to score counterfactual coherence (usually different to the counterfactual generator model)
- counterfactual_coherence_scored_by_external_model [optional, Bool] # coherence (True/ False) as judged by external_scoring_model
- counterfactual_coherence_explanation_by_external_model [optional, str] # explanation of coherence score by external_scoring_model
- counterfactual_hamming_distance [option, int] # Hamming distance from original question

- reference_model [optional, str] # model used to generate reference answers
- reference_model_answer [optional, str] # answer generated by reference model
- reference_model_explanation [optional, str] # just keep raw output.
- reference_model_explanation_confidence [optional, str] # str valued (low/ medium/ high)

- predictor_model [optional, str] # model used to predict reference model answers
- predictor_model_answer_with_explanation [optional, str] # prediction with explanation
- predictor_model_answer_without_explanation [optional, str] # prediction without explanation
- predictor_model_answer_with_explanation_justification [optional, str] # justification of prediction made with explanatnion
- predictor_model_answer_without_explanation_justification [optional, tr] # justification of prediction made without explanation
- predictor_model_answer_with_explanation_justification_confidence [optional, str] # confidence in justification...
- predictor_model_answer_without_explanation_justification_confidence [optional, str] # confidence in justiciation...

- match_with_explanation [optional, int] # 1/0 - does the prediction made with explanation match the reference model answer?
- match_without_explanation [optional, int] # 1/0 - does the prediction made without explanation match the reference model
- match_delta [optional, int] # match_with_explanation - match_without_explanation
